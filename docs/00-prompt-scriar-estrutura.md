Analise o arquivo @"docs/Online Sales Data.csv" que contém os dados de venda conforme a explicação no arquivo @docs/data-sales.md .
Gere um arquivo chamado `schema.sql` no diretório `docs` para criar um database para armazenar os dados do arquivo.

Separe as tabelas em:
* Category 
* Product
* Region* Payment Method
* Order

Gere um arquivo chamado `seeds.sql` no diretório `docs` com os comandos de `insert` nas tabelas criadas com os dados que estão no arquivo csv.
